## Challenge Overview

Investigate a seized laptop from a suspected insider threat. Perform disk forensics to recover deleted files, analyze Windows Registry for persistence mechanisms, and extract evidence of data theft.

## Learning Objectives

- Apply forensics analysis techniques
- Use professional security tools
- Understand real-world attack scenarios
- Document findings professionally

## Tools You May Need

- Autopsy or FTK Imager
- Registry Explorer
- Windows forensics toolkit
